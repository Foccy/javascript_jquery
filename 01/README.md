  - let **func=function(){};** **함수**

  - var **stringVar**='String'; **문자**
  - var **obj={};** **객체**
  - var **arr=[];** **배열**
  - var **undefinedVar=undefined;** **값 미정 default**
  - var **nullVar=null;** **빈값**
